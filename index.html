What is this?

This roblox group is run by a solo indie developer who has too many time on their hands(subject to change however). I use this group to store robux and host games. However, in the future I might even host UGC on this group.

I created this website recently for a school project. Otherwise, this website is to be used as an information hub relating to this group.
